## 🚀 Running Locally

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
cd YOUR_REPO_NAME
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm start
```

Open:

```
http://localhost:3000
```

---

## 🌐 Deployment (GitHub Pages)

The site is deployed using GitHub Pages.

To deploy updates:

```bash
npm run deploy
```

Your portfolio will automatically update at:

```
https://YOUR_USERNAME.github.io/
```

## ✏️ Updating Content

You can update your portfolio anytime by editing:


After making changes:

```bash

git push
npm run deploy
```
